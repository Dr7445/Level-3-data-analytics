For this assignment I will be using 2 datasets extracted from the Church of England data service.

Both files are currently in the .xlsx format.

The first dataset contains data on Church attendance figures by diocese over several years, the second dataset contains figures on donations given during services over several years.

I will be manipulating the data and linking it to see if I can establish a direct relationship between attendance numbers and amounts received in giving.

The following manipulation techniques will be used.


  • Cleaning the data where necessary , standardizing formats such as dates as required, ensuring consistency and maintaining the integrity of the data.
  • Filtering and sorting the data by removing columns unnecessary for this task and compiling the data so it is more suitable for the objective if required.
  • Transforming the data by creating new columns and calculating any values useful to this analysis.
  • Normalization, ensuring that the values used are consistent across the datasets, time/date formats, decimalisation etc.
  • Fuzzy matching to remove inconsistencies found in diocese names.

These steps ensure that both datasets are clean, consistent, and prepared for linking by their common attributes like diocese and year.
